# 📊 Sales Dashboard (Excel)

This project is an interactive Sales Dashboard built using Microsoft Excel.

## 🔧 Tools Used
- Excel
- Pivot Tables
- Pivot Charts
- Slicers

## 📈 Key Features
- Sales Trend Analysis
- Revenue by Country
- Top 10 Products by Revenue
- Profit Analysis
- Interactive Filters (Year, Country, Gender, Product)

## 📊 KPIs
- Total Revenue
- Total Cost
- Total Profit
- Profit Margin
- Average Order Value
